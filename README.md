# React コンポーネント練習プロジェクト

react-components-learning.md のサンプルコード集です。

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

## ファイル構成

- `src/App.tsx` - エントリポイント。各サンプルを切り替えて確認
- `src/Greeting.tsx` - サンプル1: 基本のコンポーネントとprops
- `src/Counter.tsx` - サンプル2: state とイベント
- `src/TodoList.tsx` - サンプル3: リストレンダリングとkey
- `src/Calculator.tsx` - サンプル4: 状態のリフトアップ
- `src/Settings.tsx` - サンプル5: カスタムフック
- `src/ElementDemo.tsx` - サンプル6: Component/Element/Instance体感
- `src/hooks/useLocalStorage.ts` - カスタムフック実装

## 演習問題

react-components-learning.md の「演習問題」を見て、`src/exercises/` に
自分で実装してみてください。
